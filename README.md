# 📦 ImmortalWrt for 360 V6 (IPQ60xx)

## 📝 项目背景
本固件基于 **ImmortalWrt SNAPSHOT** 编译，专为 **360 V6 (高通 IPQ60xx / aarch64)** 硬件优化。考虑到该设备的硬件性能与存储空间限制，固件设计遵循**「精简、稳定、够用」**的原则。

为满足不同程度的定制需求，本项目提供两个版本供下载：

---

## 🆕 版本一：增强版 (Enhanced) - 🌟 推荐

> **设计初衷**：
> 此前发布的最原始基础版功能较为单薄，日常使用略显不便。为了提升实用性，特此在原版基础上进行了**轻量化增强**。

### 🧩 新增功能
仅精选集成以下 **5 款** 高频实用插件，满足核心需求，不再臃肿：
- `luci-app-opkg`：增强型软件包管理界面，可视化安装更便捷
- `luci-app-fileassistant`：轻量级文件管理助手，方便配置备份与文件传输
- `luci-app-wol`：网络唤醒 (Wake-on-LAN)，远程开机
- `luci-app-autoreboot`：定时自动重启，保持系统长期稳定
- `luci-app-adguardhome`：高性能 DNS 过滤与广告拦截

### 🛑 停止增加功能的说明
考虑到 360 V6 的硬件局限性，**本版本不再考虑增加其他功能**。
目前的组合已能覆盖日常核心需求，保持系统轻量化是保证 WiFi 稳定与低延迟的关键。

---

## 🛡️ 版本二：基础版 (Original/Base)

> **设计初衷**：
> 最原始、最纯净的基础镜像，仅保留官方核心模块。

### 📋 内置功能清单
此版本未包含任何第三方扩展插件，完整保留了原生 ImmortalWrt 的基础架构：
- **🔧 系统**：系统设置、管理权限、软件包、启动项、计划任务、挂载点、LED 配置、备份更新、终端等。
- **🌐 服务**：仅预装 `HomeProxy` 作为核心网络服务支持。
- **📡 网络**：完整支持接口、无线、路由、DHCP、DNS、网络诊断及防火墙管理。

### 💡 适用人群
* 喜欢极致纯净系统，不希望有任何后台占用。
* 习惯于通过 SSH 命令行自行安装配置的高级用户。
* 仅需基础路由/代理功能，不需要广告拦截或文件管理的用户。

| :--- | :--- |

| 用户名 | `root` |
| 初始密码 | 无 (`none`) |

## 📦 文件说明
| 文件名后缀 | 适用场景 |
| :--- | :--- |
| `*-factory.ubi` | **首次刷机**：在 U-Boot 下刷入，将原厂分区格式化为 UBI。 |
| `*-sysupgrade.bin` | **日常升级**：已在 OpenWrt 系统内，通过 LuCI 界面或命令行升级。 |
| `*-uImage.itb` | **救砖/测试**：通过 TFTP 载入内存启动，**不写入闪存**，断电即还原。 |
| `.config` | **编译配置**：记录了本次编译的所有插件与内核选项，方便社区复现与调试。 |

## ⚠️ 刷机提示
1. **备份数据**：刷机前请务必通过「系统 → 备份/升级」导出当前配置。
2. **首次登录**：连接路由器 LAN 口，浏览器访问 `192.168.1.1`，用户名 `root`，密码留空。
3. **AdGuardHome 配置**：增强版用户首次进入「服务」→「AdGuardHome」需完成初始化向导，建议将上游 DNS 修改为国内公共 DNS（如 `223.5.5.5`）以获得最佳解析速度。
4. **版本选择**：新手或追求开箱即用请下载 **增强版**；极客或需要极致精简请下载 **基础版**。

> **精简，是为了更持久地稳定运行。**
